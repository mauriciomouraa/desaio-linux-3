Desafio de Projeto do Curso de Linux da Dio do Módulo:
Docker: Utilização prática no cenário de Microsserviço

Neste desafio foi possível configurar uma máquina Linux e
subir um banco Mysql em um container utilizando Docker.
Assim, como criar outros containers e assim formando um cluster.

Além disso, foi possível inserir dados em um banco com php, onde 
foi instalado um container apenas para o php
